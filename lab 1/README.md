# Лабораторная работа №1
## Моделирование принятия зачета по математике

### 📝 Описание программы

Программа моделирует процесс сдачи зачета по математике. Студенты решают квадратные уравнения и отправляют ответы преподавателю. Преподаватель проверяет все работы и составляет таблицу успеваемости.

---

### 👨‍🎓 Типы студентов

| Тип | Поведение | Вероятность успеха |
|:---:|-----------|:------------------:|
| **Хороший** | Всегда решает правильно | 100% |
| **Средний** | Решает правильно с вероятностью 70%, иначе пишет два случайных корня | 70% |
| **Плохой** | Всегда пишет два случайных корня | 0% |

---

### 🔧 Функционал программы

1. Чтение квадратных уравнений из файла `equations.txt`
2. Генерация решений каждым студентом (согласно его типу)
3. Проверка работ преподавателем
4. Подсчёт правильно решённых задач
5. Вывод таблицы успеваемости
